(ex.)

| (x0,y0) | (x1,y0) | (x2,y0) |
| (x0,y1) | (x1,y1) | (x2,y1) |
| (x0,y2) | (x1,y2) | (x2,y2) |

n = sqrt(x^2 + y^2)

se n > m -> cor preta
se n < m -> cor branca
