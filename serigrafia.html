serigrafia em papel e tecido

figurativos
::
processamento de informação RGB -- comparação entre pixels adjacentes -- mútuo canibalismo de frames

geométricos
::
equações matemáticas
f(x, y)
onde x e y
são coordenadas
do quadro de pixels

ex.
::
[ x0y0 ][ x1y0 ][ x2y0 ]
[ x0y1 ][ x1y1 ][ x2y1 ]
[ x0y2 ][ x1y2 ][ x2y2 ]
para cada pixel:
n = sqrt(x^2 + y^2)
m = 10
se n > m :: cor preta
se n < m :: cor branca

etapa de impressão
::
composição, cor e padronagem
